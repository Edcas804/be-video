# Be Video

### 📦 INSTALLATION

-   Node >= 18
-   copy `.env.dist` file.

```
cp .env.dist .env
```

```
npm install
```

### 🏃 RUN LOCALLY

```
npm run dev
```

or

```
npm run dev:host
```

### 🐍 RUN ESLINT RULES

```
npm run lint
```

### 💡 ABOUT MARVEL'S APIS KEY

The application is connecting to the marvel api to display the data, the respective keys have been added in the .env.dist file, please make proper use of them.
